Politietsregisterblade, API v. 1.0

Politietsregisterblade.dk offers the following services for free use, as long as the server load is within a reasonable level.

A note on the data structure: The central variable in the system is the card of registration (registerblad). Often, several persons is mentioned on the card. At the same time, several addresses can be written on the card. This is why addresses and persons are not directly related. The id of the card of registration (registerblad_id) is used to connect those informations.

Persons

Service http://www.politietsregisterblade.dk/api/1?type=person

Retrives a list of persons based on a specific search in specified data concerning the person.

Parameters:

  • firstnames: The first name of the person. Several conditions is separated by ";"
  • lastname: The last name of the person. Several conditions is separated by ";"
  • birthplace: The birthplace of the person
  • birthdate The date of birth of the person. The expected date format is dd-mm-yyyy
  • deathdate: The date of death of the person. The expected date format is dd-mm-yyyy
  • freetext: A free text search, similar to the freetext-service mentioned above. Several conditions is separated by ";"
Example http://www.politietsregisterblade.dk/api/1?type=person&firstnames=marie&birthdate=01-01-1903
 

Addresses

Service http://www.politietsregisterblade.dk/api/1?type=address

Retrives a list of addresses based on an id or the id of the registerblad.

Parameters:

  • id: The address id to search for
  • registerblad_id: The registerblad id to search for
Example http://www.politietsregisterblade.dk/api/1?type=address&id=1
 

Roads

Service http://www.politietsregisterblade.dk/api/1?type=road

Retrives a list of roads in Copenhagen for the period 1890 to 1924.

Parameters:

  • id: The id of the road
  • name: The name (or part of the name) of the road
Example http://www.politietsregisterblade.dk/api/1?type=road&name=gade
 

Cards of registration (registerblade)

Service http://www.politietsregisterblade.dk/api/1?type=registerblad

Retrives a list of registerblade based on the id of the registerblad.

Parameters:

  • id: The registerblad id to search for
Example http://www.politietsregisterblade.dk/api/1?type=registerblad&id=1323659
 

Free text

Service http://www.politietsregisterblade.dk/api/1?type=freetext

Based on all informations related to the card of registration, this search returns persons that matches the filters.

Parameters:

  • filter: The text to search for
Example http://www.politietsregisterblade.dk/api/1?type=freetext&filter=anders;abildgaardsgade
